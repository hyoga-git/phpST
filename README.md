# 学習ステップ（PHP → JavaScript）

本リポジトリ内で学習を進める際の推奨ステップです。  
PHP 基礎から応用、JavaScript までを無理なく進められるように構成しています。

---

## 1. PHP 基礎（言語の核）
- 基本文法
- 型
- 配列
- 制御構造（if / foreach / match）
- 関数（引数・戻り値の型）
- クラス（OOP の基礎）
- 例外
- 演算子（?? ?: ?->）

---

## 2. PHP 応用（Laravel の理解に直結）
- トレイト
- 名前空間
- セッション
- ファイル操作
- Cookie
- HTTP の基本

---

## 3. PHP 追加（余裕があれば）
- enum
- アトリビュート
- Composer オートロード
- PHP CLI の基本

---

## 4. DI / その他技術
- 依存性注入（DI）
- Service / Repository パターン
- DTrace（概要レベル）
- PIE（概要レベル）

---

## 5. Laravel 実戦
- Controller / Request / Model の流れ
- ルーティング理解
- リレーション
- CRUD の小規模実装

---

## 6. JavaScript 基礎
- 変数（let / const）
- 配列・オブジェクト
- 関数・アロー関数
- クラス
- 例外
- Promise / async 基本

---

## 7. JavaScript DOM / ブラウザ操作
- DOM 操作  
- イベント処理  
- fetch API  

---

## 8. JavaScript 応用（任意）
- モジュール（import / export）
- Node.js（CLI 実行）
- package.json / npm

---

## 補足
- PHP ⇨ JS は相性が良いため、PHP の基礎を終えてから進むと効率が高い  
- 各ステップで小さなコードを書き、実行して確認することを推奨
